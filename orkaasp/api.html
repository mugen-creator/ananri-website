API機能

🚀

RESTful API

標準的なRESTful設計で直感的に利用可能。JSON形式でのデータ交換に対応。

🔔

Webhook

リアルタイムイベント通知。成果発生、承認、否認を即座に通知。

🔐

OAuth 2.0

セキュアな認証。APIキー＋シークレットによる強固な認証。

📊

バルクAPI

大量データの一括処理。CSVエクスポート/インポート対応。

主要エンドポイント

  • GET /api/v1/reports/conversions - 成果レポート取得
  • POST /api/v1/links/create - 広告リンク生成
  • PUT /api/v1/conversions/{id}/status - 成果承認ステータス更新
  • GET /api/v1/campaigns - キャンペーン一覧取得

レート制限

APIの安定性を保つため、以下のレート制限を設けています：

  • スターター: 60リクエスト/分、10,000リクエスト/日
  • プロフェッショナル: 300リクエスト/分、100,000リクエスト/日
  • エンタープライズ: 無制限
